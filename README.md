#### Install
```
npm install
```

### Running
Start MongoDB (Windows):
```
C:\Program Files\MongoDB\Server\3.2\bin\mongod.exe
```

Start the app:
```
npm start
```

### Testing
```
mocha
```

### Linting
```
npm run linter
```
